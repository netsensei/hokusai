# Hokusai

 > “The secret to getting ahead is getting started.”  -- Mark Twain

 Hokusai is a HTML page you can a set as default start page in your browser.

## Installation

Clone or download this repo, generate assets, install in your browser.

```bash
$ git clone https://github.com/netsensei/hokusai.git hokusai
$ cd hokusai
$ npm install
$ npx mix
```

An `index.html` along with assets will be generated in the `static` directory.
In your browser settings, configure the default start page so it points to this file.
